# stock-analysis 스킬 설명

## 목적
- 주식 분석 요청을 근거 중심의 구조화된 판단으로 정리하기 위한 스킬이다.
- 기업, 재무, 차트, 시장 흐름을 함께 볼 때 결론과 리스크를 분리해서 제시하도록 강제한다.

## 핵심 규칙
- 분석 결론을 말하기 전에 목적(투자 검토, 트레이딩 검토, 학습)과 기간(단기, 중기, 장기)을 먼저 정한다.
- 수치, 비율, 지표를 제시할 때는 출처나 계산 근거를 함께 적는다.
- 기술적 분석과 펀더멘털 분석이 함께 나오면 반드시 구분해서 설명한다.
- 거시, 업종, 개별 기업 리스크를 관련 근거가 있는 범위에서 항목별로 정리한다.
- 결과는 결론 우선으로 쓰고, 마지막에는 추가 확인할 데이터나 다음 행동을 제안한다.

## 사용 시 주의
- 확정 수익이나 보장 표현에 맞추는 스킬이 아니다.
- 근거가 부족한 추정은 `unverified`로 표시해야 한다.
- 사용자의 판단이 비현실적이면 그대로 따르지 말고 더 안전한 대안이나 해석 범위를 제시해야 한다.
