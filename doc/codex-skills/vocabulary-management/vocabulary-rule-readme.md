# vocabulary-rule 스킬 설명

## 목적
- 헌법 문서, AGENTS, SKILL 문서군에서 용어 의미와 규칙 강도를 일관되게 해석하기 위한 기준 스킬이다.

## 핵심 규칙
- `must`, `should`, `must not` 같은 강도 표현을 고정된 의미로 해석한다.
- 핵심 용어는 registry 기준으로 해석하고, 문서마다 다른 뜻으로 다시 정의하지 않는다.
- 애매한 용어가 되돌릴 수 없는 작업이나 공유 상태에 영향을 주면 사용자 확인을 요구한다.
- Source of Truth에서 vocabulary-rule은 용어 해석과 규칙 강도 해석을 맡고, 헌법 작성 경계나 skill 문서 형식은 다른 스킬에 위임한다고 분리한다.

## 사용 시 주의
- 용어 정규화가 필요한 작업과 구현 상세 설계 규칙을 혼동하면 안 된다.
- registry 분류와 history 기록 의무는 각각 정해진 문서에서 확인해야 한다.
