# research 스킬 설명

## 목적
- 질문 정의, 사실과 해석 분리, 관점 비교, 불확실성 보고를 구조화하기 위한 스킬이다.

## 핵심 규칙
- 이 스킬은 질문 정의, 사실/해석 분리, 관점 비교, 불확실성 보고를 맡는다.
- 도메인별 action workflow나 MCP 파라미터 검증은 직접 맡지 않는다.
- 결론보다 먼저 질문과 근거 구조를 정리해야 한다.

## 사용 시 주의
- 리서치 구조와 실행용 워크플로 규칙을 섞으면 안 된다.
- Source of Truth에는 어떤 종류의 판단을 이 스킬이 맡는지만 남겨야 한다.
