# Purpose
- 한국어로 번역된 규칙 문서와 스킬 문서의 어휘, 식별자 토큰, 보고서 필드명, README 구조 설명의 정합성을 바로잡는다.

# Incident Summary
- 한글화 과정에서 리뷰 단계 식별자, 보고서 필드 키, 워크플로 관련 용어, README 구조 설명이 서로 다르게 번역되어 문서 간 불일치가 생겼다.

# Root Cause
- exact identifier와 일반 산문을 같은 방식으로 번역했다.
- 상위 기준 문서와 활성 어휘 레지스트리를 모두 확인하기 전에 일부 문서를 먼저 한글화했다.
- 실제 저장소 구조와 README 설명을 함께 검증하지 않았다.

# Impact
- Claude 교차 검토 단계 식별자가 충돌했다.
- 보고서 형식 키가 문서마다 달라 자동 처리와 검토 기준이 흔들렸다.
- README가 실제 스킬 폴더 구조를 잘못 설명했다.

# Changes Applied
- `AGENTS.md`, `README.md`, `CLAUDE.md`, `codex-skills/runtime-management_claude-review-runtime/SKILL.md`, `codex-skills/tool-usage-management_claude-cross-review-protocol/SKILL.md`, `codex-skills/runtime-management_work-runtime/SKILL.md`, `codex-skills/skill-management_constitution-rule/SKILL.md`, `codex-skills/action-management_automation/SKILL.md`, `codex-skills/tool-usage-management_github-mcp/SKILL.md`, `codex-skills/tool-usage-management_mariadb-mcp/SKILL.md`, `codex-skills/tool-usage-management_stock-mcp/SKILL.md`, `codex-skills/vocabulary-management_vocabulary-rule/SKILL.md`를 수정했다.
- source-review에서 `AGENTS.md`의 `runtime 규칙 갱신`이 영·한 혼용이라는 low NG가 나왔지만, 현재 워크스페이스를 지배하는 exact trigger 문구와 일치시키기 위해 유지했다.
- result-review는 `OK|no-ng|합의된 항목 생략`으로 종료됐다.
- 이후 실제 Claude 리뷰 로그 기준으로 `mariadb-mcp`, `claude-review-runtime`, `work-runtime`, `constitution-rule`에 남아 있던 용어 불일치 NG를 추가로 수정했다.
- 추가 리뷰 결과에 따라 `github-mcp`, `constitution-rule`, `README.md`에 남아 있던 잔여 NG를 다시 수정했다.

# Recurrence Prevention Rules
- exact identifier는 번역하기 전에 상위 기준 문서와 실행 중인 규칙 문서에서 표준 토큰을 먼저 확인한다.
- README 구조 설명은 실제 디렉터리 목록과 대조한 뒤에만 수정한다.
- 보고서 필드 키는 상위 문서에서 지정한 고정 키를 그대로 유지한다.
